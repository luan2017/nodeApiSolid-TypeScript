## Development of an Api SOLID.
### TypeScript + Node Js  


### Comands to compile:
- yarn 
- yarn start

### Commands to start the project from 0:
- yarn init -y
- yarn add express
- yarn add typescript ts-node-dev -D
- yarn tsc --init
- yarn add @types/express -D
- yarn add uuidv4
- yarn add nodemailer
- yarn add @types/nodemailer -D
